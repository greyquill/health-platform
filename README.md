# Greyquill Health Platform — product showcase

A static, hosted showcase + manual for the **Greyquill Health Platform**, a white-label, multi-tenant
hospital operating system built by Greyquill Software. Presented from the perspective of Greyquill
demoing the platform to a prospective hospital.

Pages:
- `index.html` — overview / value proposition
- `features.html` — comprehensive capabilities manual (by role / department)
- `interoperability.html` — FHIR R4, HL7 v2, terminology service (the coexistence layer)
- `platform.html` — architecture, RBAC, multi-tenancy, audit trail, deployment & security
- `adoption.html` — progressive, no-disruption rollout + objection handling + the case to adopt
- `demo.html` — guided demo walkthrough + request-access CTA

Plain HTML + CSS (`assets/style.css`, "Medical Blue" design system). No build step. Hosted via
GitHub Pages.

Note: this is a public marketing/manual surface. It deliberately contains **no live URLs, gate
tokens, or demo credentials** — live demo access is provisioned per hospital on request.
